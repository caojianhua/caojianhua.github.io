注意事项

如果你是在阿里云海外节点配置ss, 请务必查看此处

  • 如果服务器是专有网络, YOUR_SERVER_IP配置的应该是你的私有ip地址, 客户端配置使用公网ip地址
  • 服务器实例中需要手动添加TCP的入网ip, 并设置为允许所有ip访问 为0.0.0.0/0
  • 在网上查看教程是有遇到说需要执行 iptables -A INPUT -p tcp --dport 8778 -j ACCEPT来放开端口, 实测发现不执行该命令也一样可以使用
  • 使用chacha20的加密方式时启动提示缺少libsodium, 可以通过 apt install libsodium* 进行安装
